# 范型

## 简单范型
* Java 范型的核心概念:告诉编译器项使用什么类型，然后编译器帮你处理一切细节。
范型类
```java
public class Holder<T> {
    private T value;

    public Holder() {
    }

    public Holder(T a) {
        this.value = a;
    }

    public void set(T a) {
        this.value = a;
    }

    public T get() {
        return this.value;
    }

}
```
